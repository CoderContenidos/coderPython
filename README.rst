# Final project for coder house python course

===================
Docker Installation
===================
Pre-requisites
=================

* Docker
* Docker compose

Clone repository
================
::

    mkdir ~/projects
    cd ~/projects
    git clone git@github.com:FacundoTogliefoso/final-project.git
    cd ~/projects/final-project

Create docker image
===================
::

    docker-compose up --build
